# Logic Gates using Verilog HDL

## 📖 Project Overview
This project implements the basic logic gates using Verilog HDL.

## 🚀 Logic Gates 
- AND Gate
- OR Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- NOT Gate

## 🛠️ Tools Used
- Verilog HDL
- Xilinx Vivado

## 📂 Project Files
- allgates.v
- allgates_tb.v

## 🎯 Learning Outcome
- Learned the syntax of Verilog HDL.
- Implemented combinational logic circuits.
